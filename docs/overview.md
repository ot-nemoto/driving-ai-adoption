# AI推進 概要

## ミッション

社内におけるAI活用を推進し、業務効率化と生産性向上を実現する。

## 背景・目的

- AIツールの急速な進化により、業務への活用機会が拡大している
- 競合他社に遅れをとらないよう、組織的なAI活用の底上げが必要
- 開発効率化を軸としつつ、全社的なAI活用リテラシーの向上を目指す

## 推進体制

- 推進担当：兼務
- 対象：全社員（約50名）

## ターゲットと方針

AI推進は以下の2層構造で進める。

### Layer 1 — 開発者（メインターゲット）

| 項目 | 内容 |
|------|------|
| 目標 | 開発生産性の向上 |
| 主なツール | コーディング支援（GitHub Copilot / Cursor / Claude Code など） |
| 主なユースケース | コーディング支援、コードレビュー補助、テスト生成、ドキュメント生成 |

### Layer 2 — 経営・企画・総務・人事

| 項目 | 内容 |
|------|------|
| 目標 | ドキュメント作成・情報整理・調査業務の効率化 |
| 主なツール | 汎用AIチャット（Claude / ChatGPT など） |
| 主なユースケース | 文書作成、情報整理、調査・要約 |

## 全社共通の前提整備

推進に先立ち、以下を整備する必要がある。

- **AIツール利用ポリシー** — 利用可能なツール、情報管理のルール
- **全社周知** — 推進の目的・方針・変化の説明

## 推進ステップ（フェーズ計画）

### Phase 1 — 現状把握（現在）

全社アンケートにより、現状のAI活用状況と業務課題を把握する。

- AI利用状況（使っているツール・頻度）
- 時間がかかっている・大変な業務
- AIへの関心・不安・懸念

→ 詳細は [survey-plan.md](survey-plan.md) を参照（作成予定）

### Phase 2 — 方針策定

アンケート結果をもとに、具体的な推進施策・優先順位を決定する。

### Phase 3 以降 — 施策実行

策定した方針にもとづき、ツール導入・研修・定着支援を実施する。
