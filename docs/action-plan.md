# Role別アクションプラン

ロードマップ（[roadmap.md](./roadmap.md)）の各手法を、Role（[ai-adoption-framework.md](./ai-adoption-framework.md)）別に割り当てたものです。
各手法には **主担当**（実行責任者）と **協力**（支援・連携先）を定義します。

---

## 凡例

| 記号 | 意味 |
|---|---|
| ◎ | 主担当（実行責任者） |
| △ | 協力（支援・連携） |
