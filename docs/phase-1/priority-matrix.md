# 優先度マトリクス

全手法をインパクト × 実現容易性の2軸で評価し、優先度を分類したものです。

---

## 評価軸の定義

### インパクト
| 評価 | 基準 |
|---|---|
| 高 | 3つ以上の骨子に対応する、または手薄な骨子（⑧⑨⑩）をカバーする、または他の手法の前提になる |
| 中 | 2つの骨子に対応する |
| 低 | 1つの骨子のみ、または効果が限定的 |

### 実現容易性
| 評価 | 基準 |
|---|---|
| 高 | 今すぐ着手可能。大きな予算・承認・組織変更不要 |
| 中 | 一定の準備・予算申請・調整が必要 |
| 低 | 大きな投資・組織変更・長いリードタイムが必要 |

---

## 優先度分類の定義

| 分類 | 条件 |
|---|---|
| 🟢 今すぐやる | インパクト 高/中 × 実現容易性 高 |
| 🟡 計画してやる | インパクト 高/中 × 実現容易性 中/低 |
| ⚪ 後回し | インパクト 低、または実現容易性 低 × インパクト 低/中 |
