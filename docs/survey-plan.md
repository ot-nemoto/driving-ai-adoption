# 全社AIアンケート 設計書

## 目的

- 社内のAI活用の現状を把握する
- 業務課題・活用余地を特定し、推進施策の優先順位付けに活用する

## 方針

- **所要時間：5〜7分以内**（回収率を高めるため）
- Google Formsのセクション分岐機能を使い、職種別に設問を出し分ける
- 選択式を中心とし、自由記述は最小限に抑える

---

## フォーム構成

```
セクション1：共通設問（全員）
  ↓ 職種で分岐
セクション2A：開発者向け設問
セクション2B：非開発者向け設問
  ↓
セクション3：共通設問（全員）
```

---

## 設問詳細

### セクション1：共通設問（全員）

**Q1. あなたの職種を教えてください**（単一選択）
- エンジニア / 開発者
- 企画 / ディレクター
- 営業
- 総務 / 人事 / 経理
- 経営 / マネージャー
- その他

> ※ この回答をもとに、次の設問が切り替わります

**Q2. 現在、業務でAIツールを使っていますか？**（単一選択）
- ほぼ常時使っている（業務の大半で活用）
- 定期的に使っている（週に数回）
- たまに使っている（月に数回程度）
- ほとんど使っていない
- まったく使っていない

**Q3. 使っているAIツールをすべて選んでください**（複数選択）
- ChatGPT
- Claude
- GitHub Copilot
- Cursor
- Gemini
- Microsoft Copilot（M365）
- その他（自由記述）
- 使っていない

---

### セクション2A：開発者向け設問

**Q4. 各開発フェーズにおける現在のAI活用度を教えてください**（マトリクス・5段階）

行（フェーズ）：
- 要件定義
- 基本設計
- 詳細設計
- 実装（コーディング）
- テスト
- コードレビュー
- ドキュメント作成
- リリース・デプロイ作業

列（活用度）：
1. 使っていない
2. たまに使う
3. 定期的に使う
4. よく使う
5. ほぼAIに任せている

**Q5. 各開発フェーズにおいて、将来的にどこまでAIに任せられると思いますか？**（マトリクス・5段階）

> 行はQ4と同じ

列（活用余地）：
1. AIは使えない・使うべきでない
2. 補助的に使える程度
3. 半分程度はAIに任せられる
4. 大部分はAIに任せられる
5. ほぼすべてAIに任せられる

**Q6. AIを使いにくい・使えないと感じる場面や理由があれば教えてください**（自由記述・任意）

---

### セクション2B：非開発者向け設問

**Q4. 各業務における現在のAI活用度を教えてください**（マトリクス・5段階）

行（業務種別）：
- 文書・資料作成
- 情報収集・調査・まとめ
- データ整理・集計
- メール・連絡文の作成
- 会議の準備・議事録作成
- その他の定型業務

列（活用度）：Q4（開発者向け）と同じ

**Q5. AIを使いにくい・使えないと感じる場面や理由があれば教えてください**（自由記述・任意）

---

### セクション3：共通設問（全員）

**Q7（開発者はQ7、非開発者はQ6）. AIの活用について、期待していることや不安なことがあれば教えてください**（自由記述・任意）

## 設問数サマリー

| 対象 | 設問数 | 想定所要時間 |
|------|--------|------------|
| 開発者 | 7問（うちマトリクス2問） | 約5〜7分 |
| 非開発者 | 5問（うちマトリクス1問） | 約3〜5分 |

---

## Google Forms 設定メモ

- セクション1のQ1（職種）の回答に応じてセクション2A/2Bに分岐させる
- Q3（ツール選択）は「Q2でほとんど/まったく使っていない」を選んだ場合はスキップさせる（分岐設定）
- 自由記述はすべて「任意回答」にする
- 回答は匿名可（氏名不要）とし、回収率を高める
