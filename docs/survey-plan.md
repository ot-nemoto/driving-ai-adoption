# 全社AIアンケート 設計書

## 目的

- 社内のAI活用の現状を把握する
- 業務課題・活用余地を特定し、推進施策の優先順位付けに活用する

## 分析の考え方

アンケート結果から以下の3つを可視化する。

```
現状の活用度（Q4）
　↕ この差 = 障害によって達成できていない部分（Q6で原因を特定）
最大の活用余地（Q5）
　↕ この差（100%との差） = AIがどうしても介入できない部分（Q7で理由を特定）
100%
```

## 方針

- **所要時間：5〜7分以内**（回収率を高めるため）
- Google Formsのセクション分岐機能を使い、職種別に設問を出し分ける
- 選択式を中心とし、自由記述は最小限に抑える

---

## フォーム構成

```
セクション1：共通設問（全員）
  ↓ 職種で分岐
セクション2A：開発者向け設問
セクション2B：非開発者向け設問
  ↓
セクション3：共通設問（全員）
```

---

## 設問詳細

### セクション1：共通設問（全員）

**Q1. あなたの職種を教えてください**（単一選択）
- エンジニア / 開発者
- 企画 / ディレクター
- 営業
- 総務 / 人事 / 経理
- 経営 / マネージャー
- その他

> ※ この回答をもとに、次の設問が切り替わります

**Q2. 現在、業務でAIツールを使っていますか？**（単一選択）
- ほぼ常時使っている（業務の大半で活用）
- 毎日使っている（業務の一部で活用）
- たまに使っている（週に数回程度）
- ほとんど使っていない

**Q3. 使っているAIツールをすべて選んでください**（複数選択）
- ChatGPT
- Claude
- GitHub Copilot
- Cursor
- Gemini
- Microsoft Copilot（M365）
- その他（自由記述）
- 使っていない

---

### セクション2A：開発者向け設問

**Q4. 各タスクにおける現在のAI活用度を教えてください**（マトリクス・5段階）

行（タスクタイプ）：

＜共通タスク＞
- テキスト要約・整理（議事録作成、ヒアリング内容まとめなど）
- 文書・資料作成（仕様書、設計書、提案書など）
- 調査・情報収集（技術調査、ライブラリ比較など）
- アイデア出し・壁打ち（設計案、代替案の検討など）

＜開発者向けタスク＞
- コード生成（新規実装、ボイラープレート生成など）
- コードレビュー・デバッグ（レビューコメント生成、バグ原因調査など）
- テストコード生成・テスト設計
- テスト実行・結果分析（テスト失敗の原因調査、カバレッジ改善など）
- デプロイ・リリース（CI/CD設定、リリーススクリプト、リリースノート作成など）
- 運用・保守（ログ分析、障害調査、パフォーマンス改善など）

列（活用度）：
1. 0%（まったく使っていない）
2. 25%程度（補助的に少し使っている）
3. 50%程度（半分程度使っている）
4. 75%程度（大部分で使っている）
5. 100%程度（ほぼすべてAIで対応）

**Q5. 各タスクにおいて、将来的にどこまでAIに任せられると思いますか？**（マトリクス・5段階）

> 行はQ4と同じ・列はQ4と同じ

**Q6. AIをもっと活用できない理由として当てはまるものを選んでください**（複数選択・任意）

- スキル・知識が不足している
- 使えるツール・環境が整っていない（ライセンス・社内ルールがない）
- AIの品質・精度への不安
- セキュリティ・情報漏洩への懸念
- 試す時間・余裕がない
- その他（自由記述）

**Q7. AIをどうしても使えない・使うべきでない理由があれば選んでください**（複数選択・任意）

- 顧客への説明責任・承認が必要
- 法的・コンプライアンス上の制約
- 機密情報のため外部サービスに送れない
- 人間の判断・創造性が不可欠
- その他（自由記述）

### セクション2B：非開発者向け設問

**Q4. 各タスクにおける現在のAI活用度を教えてください**（マトリクス・5段階）

行（タスクタイプ）：

＜共通タスク＞
- テキスト要約・整理（議事録作成、ヒアリング内容まとめなど）
- 文書・資料作成（報告書、提案書、マニュアルなど）
- 調査・情報収集（市場調査、情報まとめなど）
- アイデア出し・壁打ち（企画案、改善案の検討など）

＜非開発者向けタスク＞
- データ整理・集計（表、レポート作成など）
- メール・連絡文の作成
- 会議の準備・ファシリテーション

列（活用度）：Q4（開発者向け）と同じ

**Q5. 各タスクにおいて、将来的にどこまでAIに任せられると思いますか？**（マトリクス・5段階）

> 行はQ4と同じ・列はQ4と同じ

**Q6. AIをもっと活用できない理由として当てはまるものを選んでください**（複数選択・任意）

> 選択肢はセクション2A Q6と同じ

**Q7. AIをどうしても使えない・使うべきでない理由があれば選んでください**（複数選択・任意）

> 選択肢はセクション2A Q7と同じ

**Q8. AIの活用について、期待していることや不安なことがあれば教えてください**（自由記述・任意）

## Google Forms 設定メモ

- セクション1のQ1（職種）の回答に応じてセクション2A/2Bに分岐させる
- Q3（ツール選択）は「Q2でほとんど使っていない」を選んだ場合はスキップさせる（分岐設定）
- Q6・Q7・Q8はすべて任意回答にする
- 回答は匿名可（氏名不要）とし、回収率を高める
